# Resume-Ready Bullet Points

## General Full Stack / Python / GenAI

- Designed and developed a full-stack AI-powered Job Application Assistant using Python, FastAPI, Streamlit, and local LLM integration through Ollama.

- Built modular REST APIs for ATS-friendly resume optimization, cover letter generation, interview question generation, ATS analysis, and AI-powered career assistance.

- Integrated multi-model local GenAI inference using Ollama with support for Llama 3, Mistral, Phi-3, and DeepSeek Coder for task-specific AI workflows.

- Engineered prompt-driven AI workflows for contextual resume tailoring, interview preparation, ATS keyword analysis, and conversational career guidance.

- Developed an interactive Streamlit frontend with file upload support, model selection, downloadable reports, and ChatGPT-style AI chat interface.

- Implemented resume parsing for PDF, DOCX, and TXT documents using PyPDF2 and python-docx for automated AI workflow ingestion.

- Built structured ATS resume analysis with match percentage scoring, missing keyword extraction, and improvement recommendation generation.

- Added session memory and generated output history using Streamlit session state for enhanced conversational and workflow continuity.

- Containerized the application using Docker and Docker Compose for reproducible local deployment.

- Created Kubernetes deployment manifests including namespace, deployment, service, configmap, and ingress-ready architecture for orchestration readiness.

# Job Application AI Assistant - Project Planner

## Project Overview

Build a complete AI-powered Job Application Assistant using local GenAI models with Python.

Primary goal:
Create a portfolio-ready GenAI application that helps users with:

- Resume optimization
- ATS analysis
- Cover letter generation
- Interview preparation
- Career AI chat support

Architecture goals:

- Local AI inference
- API-first backend
- polished frontend UI
- containerized deployment
- Kubernetes readiness
- scalable modular design
